# Support

Use GitHub Issues for reproducible bugs and feature requests. This project is maintained on a best-effort basis and does not provide emergency recovery or guaranteed response times.

Before requesting support:

1. Back up important recordings.
2. Do not share rclone configuration or credentials.
3. Run `01_检查电脑环境.bat`.
4. Review `%USERPROFILE%\ScreenAgent\logs\report.txt`.
5. Redact titles, paths, account names, and remote URLs.

Credential exposure and unintended file movement are security reports; follow `SECURITY.md` instead of opening a detailed public Issue.
