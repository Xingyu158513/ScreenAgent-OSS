# ScreenAgent 交付验收清单

## 交付者

- [ ] ZIP 文件名包含版本号。
- [ ] 提供 ZIP 的 SHA-256。
- [ ] ZIP 来自干净的发行目录，不包含密码、rclone 配置、个人日志或视频。
- [ ] 所有 PowerShell 文件通过语法检查。
- [ ] 默认清理策略是 `move_after_verified_upload`。
- [ ] 已测试上传失败、验证失败和大小不一致时不删除本地文件。
- [ ] 已测试远程重名时不会覆盖。
- [ ] 已测试中文用户名、中文标题和中文云端目录。

## 接收者

- [ ] ZIP 的 SHA-256 与交付者提供的值一致。
- [ ] 完整解压 ZIP。
- [ ] 阅读 `开始使用.html` 和 `docs\安装前准备.md`。
- [ ] 运行 `检查环境.bat`。
- [ ] 只从官方页面下载 OBS 和 rclone。
- [ ] 使用坚果云第三方应用密码，不使用登录密码。
- [ ] 安装后确认 OBS 预览正常、录制目录正确。
- [ ] 已确认界面只提供“验证后移动”与“保留本地”，不存在永久删除模式。
- [ ] 在真实重要录制前完成一次短视频上传测试。

## 安装后验证命令

```powershell
Get-ScheduledTask -TaskName "ScreenAgent-AutoUpload"
Get-Content "$env:USERPROFILE\ScreenAgent\config\config.json" -Raw
Get-Content "$env:USERPROFILE\ScreenAgent\logs\report.txt" -Tail 30
Get-ChildItem "$env:USERPROFILE\ScreenAgent\recordings\raw" -File
Get-ChildItem "$env:USERPROFILE\ScreenAgent\recordings\uploaded" -File
```

## 回退原则

发现异常时先停止计划任务，不要删除 raw 文件：

```powershell
Stop-ScheduledTask -TaskName "ScreenAgent-AutoUpload"
```

确认问题解决后再启动：

```powershell
Start-ScheduledTask -TaskName "ScreenAgent-AutoUpload"
```
