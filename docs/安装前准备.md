# 安装前准备

安装 ScreenAgent 前，请先准备好下面几样东西。普通用户只需要按顺序检查即可。

## 1. OBS Studio

OBS Studio 用于录屏。

请从 OBS 官方网站下载并安装：

https://obsproject.com/

安装完成后，ScreenAgent 会自动检测常见安装路径。如果没有检测到，配置向导会提示你手动选择 `obs64.exe` 的位置。

常见路径是：

```text
C:\Program Files\obs-studio\bin\64bit\obs64.exe
```

## 2. rclone

rclone 用于把录屏文件上传到坚果云或其他 WebDAV 云盘。

如果你的 ScreenAgent 压缩包里已经内置了 `rclone.exe`，你不需要单独下载。

如果压缩包里没有内置 rclone，请从 rclone 官方网站下载：

https://rclone.org/downloads/

建议放到：

```text
C:\Tools\rclone\rclone.exe
```

配置向导也会自动检测系统 PATH 里的 `rclone.exe`。

## 3. 坚果云账号

如果你选择“坚果云 WebDAV”模式，需要一个可以正常登录的坚果云账号。

坚果云官网：

https://www.jianguoyun.com/

## 4. 坚果云第三方应用密码

坚果云 WebDAV 不建议直接使用登录密码。你需要在坚果云网页端生成“第三方应用密码”。

大致步骤：

1. 登录坚果云网页版。
2. 进入账号设置。
3. 找到“安全选项”或“第三方应用管理”。
4. 添加一个应用，例如命名为 `ScreenAgent`。
5. 复制生成的应用密码。
6. 安装 ScreenAgent 时，把这个应用密码填入配置向导。

ScreenAgent 不会把应用密码写入自己的 `config.json` 或安装报告。密码会交给 rclone 以加密/混淆方式保存到 rclone 自己的配置中。

## 5. 坚果云客户端（可选）

坚果云客户端不是必须的。

如果你希望在 Windows 资源管理器里更方便地查看云端文件，可以安装坚果云客户端。

但 ScreenAgent 上传文件不依赖坚果云客户端，只依赖 WebDAV 和 rclone。
