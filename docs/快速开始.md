# ScreenAgent 快速开始

## 安装前

1. 安装 OBS Studio。
2. 准备 rclone；如果只保存本地，可以暂时不安装。
3. 使用坚果云时，先在坚果云网页生成“第三方应用密码”。
4. 双击压缩包中的 `检查环境.bat`。

官方入口：

- OBS Studio：https://obsproject.com/download
- rclone：https://rclone.org/downloads/
- 坚果云第三方应用授权：https://help.jianguoyun.com/?p=2064

也可以双击根目录中的 `获取依赖.bat` 打开上述官方页面。

## 安装

1. 完整解压 ZIP，不要直接在压缩包预览窗口中运行。
2. 双击 `install.bat`。
3. 输入产品显示名称，默认是“ScreenAgent”。
4. 选择坚果云、通用 WebDAV 或只保存本地。
5. 选择本地处理策略。建议使用默认安全模式。
6. 按向导确认 OBS 录制目录。

安装完成后，桌面会出现：

- `启动录制-产品名`
- `查看 产品名 日志`

## 第一次录制

1. 打开 OBS，确认预览不是黑屏。
2. 确认 OBS 的录制路径是 `%USERPROFILE%\ScreenAgent\recordings\raw`。
3. 双击桌面的“启动录制”快捷方式。
4. 输入标题、分类和主题，选择清理策略。
5. OBS 自动开始录制。
6. 在 OBS 中点击“停止录制”。不要直接强制结束 OBS 进程。

后台程序会等待文件停止写入，然后上传并验证。

## 查看结果

双击桌面的“查看日志”快捷方式，或查看：

```text
%USERPROFILE%\ScreenAgent\logs\report.txt
```

默认安全模式下，上传验证成功的视频会移动到：

```text
%USERPROFILE%\ScreenAgent\recordings\uploaded
```

失败的视频会保留在 `recordings\raw`，等待后续重试。
