# ScreenAgent 1.1.0

状态：公开候选版

## 重点变化

- 移除自动永久删除，公开版只允许验证后移动或保留本地。
- 旧删除配置自动降级为验证后移动。
- WebDAV 密码通过标准输入交给 `rclone obscure -`，不再作为明文命令行参数。
- 把路径、凭据和本地移动边界抽成可测试安全模块。
- 卸载器不再提供递归删除全部用户数据的入口。
- 移除登录常驻扫描；录制时只运行有期限的会话 worker，遗留文件由独立的一次性恢复命令处理。
- 旧计划任务迁移与运行时归档逻辑分离。

## 自动验证

- PowerShell 文件语法解析。
- 未验证不移动。
- 验证后移动且不覆盖已有文件。
- 保留本地模式。
- 路径逃逸拒绝。
- 旧删除模式安全降级。
- 密码标准输入边界。
- 卸载不删除用户数据。

## 已知限制

- 需要用户安装并配置 OBS Studio。
- 云端模式依赖 rclone 和 WebDAV 服务。
- WebDAV 同名同大小验证不是逐字节哈希证明。
- 旧计划任务迁移和真实 WebDAV 集成仍需要在隔离 Windows 测试机上人工验收。
